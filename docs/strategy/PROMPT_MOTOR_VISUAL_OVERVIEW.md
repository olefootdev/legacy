# Motor visual: Babylon + Yuka (OLEFOOT)

Briefing de implementação do cliente de partida: simulação tática (`src/simulation/`, `src/agents/`, `src/formation/`, `src/tactics/`), snapshot `MatchTruth` (`src/bridge/matchTruthSchema.ts`), render `src/render-babylon/`.

**Admin, zonas e padrões táticos** (cadastro staff, BRO, catálogo de zonas) estão especificados em [`docs/ADMIN_TACTICAL_BRO.md`](./docs/ADMIN_TACTICAL_BRO.md) e tipados em `src/field-schema/`. O app de jogo **não** inclui telas de cadastro tático para o público.
